# PE library

This project was organized while studying windows pe file.

- PE structure view (exe, dll)
  - PE headers
  - RVA
  - Export, import
  - code, data sections
  - Base Relocation
  - DLL basic
  - DLL delayed load section(.didat)
  - .rsrc
  - SEH (x86, x64) .pdata
- PE resource extrator
- PDB parser
- DLL injector
- API hooker
- Disassembler
- Debugger
